Written as part of a project for my Evolutionary Computing class, this Pacman agent uses a Goza-tree style algorithm to determine the next best possible action to take.

Developed for a local coffee shop, Well Red, this web app allows users to browse and search the shop's current book inventory. Users can also reserve books for pickup at the shop, or request books that are not in stock. Admins are able to manage users, manage reservations and requests, and update the inventory.

Notably, the site is connected to the shop's Square account so that the inventory is automatically updated when books are sold. This project also includes a barcode scanner that allows admins to quickly scan books and add them to the inventory. It also allows admins to update the Square inventory from the physical inventory.
